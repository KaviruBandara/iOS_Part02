## Gamification Features

### Points System

**How Points Work:**
- Earn points by completing tasks
- Different tasks = different points
- Points determine your level
- Points determine leaderboard rank

**Point Values:**
- **Low Priority**: 5 points
- **Medium Priority**: 10 points  
- **High Priority**: 20 points
- **Custom**: Varies by task

**Example Tasks:**
- "Take out trash": 5 points
- "Wash dishes": 15 points
- "Clean bathroom": 25 points

### Streak System

**What is a Streak?**
- Consecutive days of task completion
- Builds when you complete at least one task per day
- Resets if you miss a day

**Why Streaks Matter:**
- Unlock streak badges
- Motivate daily participation
- Track consistency

**Streak Badges:**
- ğŸ”¥ 3-Day Streak
- ğŸ”¥ Week Warrior (7 days)
- ğŸ”¥ Fortnight Force (14 days)

- ğŸ”¥ Monthly Master (30 days)

### Level System

**Leveling Up:**
- Start at Level 1
- Every 100 points = new level
- No level cap
- Level displayed on profile

**Level Benefits:**
- Bragging rights
- Visual progress tracking
- Sense of achievement

### Badge System

**Types of Badges:**

## Troubleshooting

### Common Issues

**Q: I can't complete a task**
- A: Make sure you've claimed it first
- Only claimed tasks can be completed

**Q: My points didn't increase**
- A: Check if task was already completed
- Refresh by switching tabs

**Q: Leaderboard doesn't show me**
- A: Ensure you've completed at least one task
- Try switching users and back

**Q: Lost my streak**
- A: Streaks reset if no tasks completed for a day
- Start building a new streak

**Q: Badge didn't unlock**
- A: Check requirements carefully
- Some badges have hidden criteria

### Reset Options

**If app behaves strangely:**

1. **Switch profiles**: Log out and back in
2. **Reset daily tasks**: Settings â†’ Reset Daily Tasks
3. **Full reset**: Settings â†’ Reset All Data (âš ï¸ deletes everything)
